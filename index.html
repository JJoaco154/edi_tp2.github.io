Programación web básica

¿Sobre qué quiere aprender?


El Lenguaje de Marcado de Hipertexto (HTML) es el código que se utiliza para estructurar y desplegar una página web y sus contenidos. Por ejemplo, sus contenidos podrían ser párrafos, una lista con viñetas, o imágenes y tablas de datos. HTML no es un lenguaje de programación, es un lenguaje de marcado que define la estructura de tu contenido en la pagina web, HTML consiste en una serie de elementos que se usaran para encerrar diferentes partes del contenido. En este "lenguaje" las etiquetas arrancan con una etiqueta de apertura y termina con una de cierre, dentro de las las etiquetas va al contenido (información que queremos que se vea).


CSS, o Cascading Style Sheets, es un lenguaje de estilos utilizado para describir la presentación de documentos HTML. Permite controlar la apariencia visual de las páginas web, como colores, fuentes, márgenes y disposición de los elementos. Los selectores se utilizan para identificar los elementos HTML a los que se aplicarán los estilos, y las propiedades determinan cómo se verán esos elementos, especificando características como el color o el tamaño de la fuente. La cascada permite que los estilos se apliquen en un orden determinado, y la especificidad decide qué estilos prevalecen en caso de conflicto. Además, las media queries permiten ajustar los estilos según el tamaño de la pantalla, facilitando el diseño responsivo. CSS se puede incluir en un documento HTML de tres maneras: de forma inline, interna o externa. Es una herramienta clave para crear interfaces atractivas y funcionales en la web.


JavaScript es un lenguaje de programación creado en 1995. Se usa para hacer que las páginas web sean interactivas. Funciona en todos los navegadores y también en servidores con Node.js. Permite cambiar el contenido de una página sin recargarla, manejar eventos como clics y actualizar datos en segundo plano. JavaScript usa variables (var, let, const), funciones (tradicionales y flecha), objetos (con propiedades y métodos) y arreglos (listas de elementos). También tiene condicionales y bucles para controlar el flujo del programa. Es un lenguaje muy flexible y esencial para el desarrollo web moderno.
